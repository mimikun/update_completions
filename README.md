# update_completions

## Requirements

- [fisher](https://github.com/jorgebucaran/fisher)
- [pueue](https://github.com/Nukesor/pueue)

## Install

### Bash

```shell
git clone https://github.com/mimikun/update_completions.git
cd update_completions
make install
```

### Fish

```shell
fisher install mimikun/update_completions
```
